# Real-World Data Project: Retail Sales Analysis & Prediction 

## Overview
This capstone project demonstrates an end-to-end data science pipeline applied to the retail domain. It encompasses data generation, exploratory data analysis (EDA), time-series visualization, and predictive modeling to extract actionable business insights from daily sales data.

## Key Features
* **Domain Selection:** Focused on Retail, analyzing daily sales, marketing spend, and discount strategies over a 365-day period.
* **Exploratory Data Analysis:** Aggregated and visualized revenue by product category and implemented a 14-day rolling average to analyze long-term sales trends.
* **Predictive Modeling:** Built a Multiple Linear Regression model using `scikit-learn` to predict future daily sales based on marketing budgets and promotional discounts, evaluating performance using the $R^2$ metric.
* **Executive Dashboard:** Designed a cohesive 3-panel visualization summarizing categorical revenue, time-series trends, and model accuracy.
